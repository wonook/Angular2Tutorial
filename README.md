# Angular2 on Rails

## Technologies used: 
- `Ruby on Rails`, 
- `Angular2 with Typescript and SystemJS`, 
- `Gulp` for asset pipeline, 
- `npm` for managing front-end libraries.

## How to run

1. Install node, chruby, ruby-2.3.0
1. run `npm install`
1. run `bundle`
1. `rake db:create db:migrate`
1. run `rails s`
1. run `gulp`
1. Enjoy!
